विंडोज पर एआई कार्यों को नियंत्रित करने के लिए एक सरल इंटरफ़ेस (सिंगल-टैब MAUI कंट्रोल सरफेस) और एक स्थानीय डेमॉन, जो बेहतर प्रदर्शन प्रदान करता है - सुचारू संचालन, कम रुकावट, और कम तापमान वृद्धि।

यह प्रणाली लीज़गेट की परिचालन विशेषताओं (स्पष्ट नियंत्रण, सीमित निष्पादन, निश्चित कारण, और अवलोकन योग्य स्थिति) को बरकरार रखती है, लेकिन इसे घरेलू कंप्यूटर के उपयोग के लिए अनुकूलित किया गया है।

## परियोजनाएं।

| परियोजना। | विवरण। |
| ज़रूर, मैं आपकी मदद कर सकता हूँ। कृपया वह अंग्रेजी पाठ प्रदान करें जिसका आप हिंदी में अनुवाद करवाना चाहते हैं। | कृपया वह अंग्रेजी पाठ प्रदान करें जिसका आप हिंदी में अनुवाद करवाना चाहते हैं। मैं उसका सटीक और उचित अनुवाद करने के लिए तैयार हूं। |
| `src/LeaseGateLite.Contracts` | साझा डेटा ट्रांसफर ऑब्जेक्ट्स (डीटीओ) और एनम (net9.0 + net10.0) |
| `src/LeaseGateLite.Daemon` | `localhost:5177` पर स्थित एक स्थानीय एपीआई डेमॉन, जो वास्तविक विंडोज सिस्टम के आंकड़ों को प्रदर्शित करता है। |
| `src/LeaseGateLite.App` | एक ही टैब में काम करने वाला MAUI कंट्रोल पैनल (विंडोज/एंड्रॉइड/आईओएस/मैक कैटलिस्ट के लिए)। |
| `src/LeaseGateLite.Tray` | विंडोज सिस्टम ट्रे के लिए सहायक उपकरण। |
| `tests/LeaseGateLite.Tests` | 178 यूनिट परीक्षण (कॉन्फ़िगरेशन सत्यापन, सिमुलेशन, निदान)। |

## दौड़ो।

1) डेमॉन शुरू करें:

```powershell
dotnet run --project src/LeaseGateLite.Daemon
```

2) MAUI एप्लिकेशन शुरू करें (विंडोज):

```powershell
dotnet build src/LeaseGateLite.App -f net10.0-windows10.0.19041.0
dotnet run --project src/LeaseGateLite.App -f net10.0-windows10.0.19041.0
```

## एक क्लिक में पैकेजिंग और इंस्टॉलेशन (विंडोज के लिए)।

एक रिलीज़ पैकेज (पोर्टेबल ज़िप फ़ाइल + SHA256 चेकसम) बनाएं:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/package-v0.1.0.ps1
```

स्थानीय रूप से पैकेज किए गए फ़ाइल से स्थापित करें:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/install-local.ps1 -EnableAutostart
```

स्थापना के बाद का व्यवहार:
- डेमॉन तुरंत शुरू हो जाता है और इसे कॉन्फ़िगर किया जा सकता है ताकि यह लॉग इन करने पर भी शुरू हो।
- नियंत्रण पैनल अपने आप खुल जाता है और कनेक्ट हो जाता है।
- "बैलेंस्ड" डिफ़ॉल्ट सेटिंग है; लैपटॉप जैसे हार्डवेयर के लिए, पहली बार सेटअप करते समय "शांत" मोड की सिफारिश की जाती है (लेकिन यह अनिवार्य नहीं है)।

## एक टैब वाला लेआउट।

एकल टैब/पृष्ठ: **नियंत्रण**

- **शीर्ष भाग:** स्थिति संकेतक, मोड चयनकर्ता, अंतिम बिंदु, त्वरित क्रियाएं (शुरू करें, रोकें, लागू करें, निदान निर्यात करें)।
- **बायां कॉलम:** ऑडिट करने योग्य चेकलिस्ट (कार्ड पर जाएं)।
- **दायां कॉलम:** चेकलिस्ट अनुभागों से मेल खाने वाले क्रमबद्ध नियंत्रण कार्ड।

प्रत्येक कार्ड में निम्नलिखित जानकारी शामिल है: वर्तमान मूल्य, संक्षिप्त अर्थ, नियंत्रण, प्रभाव का पूर्वावलोकन, और एक विवरणिका।

## लेखा परीक्षा की जाँच सूची।

| अनुभाग। | नियंत्रण। |
| ज़रूर, मैं आपकी मदद कर सकता हूँ। कृपया वह अंग्रेजी पाठ प्रदान करें जिसका आप हिंदी में अनुवाद करवाना चाहते हैं। | ज़रूर, मैं आपकी मदद कर सकता हूँ। कृपया वह अंग्रेजी पाठ प्रदान करें जिसका आप हिंदी में अनुवाद करवाना चाहते हैं। |
| **A) Service** | कनेक्ट करें, शुरू/बंद करें/पुनः आरंभ करें, संस्करण + अपटाइम, कॉन्फ़िगरेशन फ़ाइल का स्थान, रीसेट करें। |
| **B) Live status** | गर्मी का स्तर (शांत/गर्म/तेज़), सक्रिय कॉल की संख्या, कतार की गहराई, सीपीयू का उपयोग प्रतिशत, रैम का उपयोग प्रतिशत। |
| **C) Core throttling** | अधिकतम समवर्तीता, इंटरैक्टिव रिजर्व, पृष्ठभूमि सीमा, कूलडाउन। |
| **D) Adaptive tuning** | नरम/कठोर सीमाएं, पुनर्प्राप्ति दर, सुचारू बनाना। |
| **E) Request shaping** | अधिकतम आउटपुट/प्रॉम्प्ट सीमा, अतिप्रवाह व्यवहार, पुनः प्रयास नीति। |
| **F) Rate limiting** | अनुरोध/प्रति मिनट, टोकन/प्रति मिनट, अधिकतम अनुमत डेटा प्रवाह। |
| **G) Presets** | शांत (लैपटॉप), संतुलित, प्रदर्शन (डेस्कटॉप)। |
| **H) Diagnostics** | निर्यात निदान, घटना लॉग का अंतिम भाग, प्रतिलिपि स्थिति का सारांश। |

## टिप्पणियाँ।

- "लाइट" संस्करण जानबूझकर जटिल प्रशासनिक सुविधाओं (अनुमोदन, हस्ताक्षर, रसीदें) को शामिल नहीं करता है।
- यह प्रोग्राम विंडोज सिस्टम के वास्तविक मापदंडों (सीपीयू प्रदर्शन काउंटर के माध्यम से, और ग्लोबलमेमोरीस्टेटएक्स के माध्यम से रैम) को पढ़ता है और कतार के दबाव की गतिशीलता का अनुकरण करता है।
- परीक्षणों में, "फेकसिस्टममेट्रिक्स" नामक एक प्रदाता का उपयोग निर्भरता इंजेक्शन के माध्यम से किया जाता है, ताकि हार्डवेयर से स्वतंत्र और निश्चित सत्यापन किया जा सके।
