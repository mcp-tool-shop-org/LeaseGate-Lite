Welcome to the LeaseGate-Lite handbook.

LeaseGate-Lite is a one-tab MAUI control surface and local daemon for throttling AI workloads on Windows. It keeps LeaseGate's operational feel — explicit control, bounded execution, deterministic reasons, observable status — trimmed to home-PC scope.

## What's inside

- **[Getting Started](/LeaseGate-Lite/handbook/getting-started/)** — Install, run the daemon, and launch the control panel
- **[Daemon API](/LeaseGate-Lite/handbook/daemon-api/)** — All REST endpoints on localhost:5177
- **[Control Panel](/LeaseGate-Lite/handbook/control-panel/)** — One-tab layout, audit checklist, and presets
- **[Configuration](/LeaseGate-Lite/handbook/configuration/)** — Throttling, rate limiting, and adaptive tuning
